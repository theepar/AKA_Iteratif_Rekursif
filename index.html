Bedah Logika

1. Modulo ( % )

Operasi sisa bagi menentukan apakah suatu bilangan ganjil (1) atau genap (0).

19 % 2 = 1 (Bit akhir)

2. Division ( / )

Membagi bilangan dengan 2 (Integer Division) sama dengan menggeser bit ke kanan (Right Shift).

Math.floor(19 / 2) = 9

3. Konkatenasi

Pada metode iteratif, urutan penggabungan string harus dibalik karena kita mendapatkan LSB lebih dulu.

biner = sisa + biner
